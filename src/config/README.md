# Config Folder

Aquí se guardan los archivos de configuración del bot.

- `config.sample.yaml`: ejemplo de configuración con claves y parámetros ficticios.
- `config.yaml`: configuración real que debe crear el usuario (basada en el sample).

Por seguridad:
- Nunca subas `config.yaml` a repositorios públicos.
- Define tu API key de OpenAI a través de variables de entorno,
  tal como se indica en `config.sample.yaml`.
